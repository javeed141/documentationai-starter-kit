---
title: Available Views for Product Data
---

The `view` parameter in the API determines which fields are returned by your data request and the order in which they're shown.  It also determines if multi-valued fields will flatten or expand their individual elements into multiple entries.

Note that each `view` is available in either `json` or `csv` format.  The `format` parameter you use will determine which one is used.

Available views for product data:

* [products_all](https://java.documentationai.com/docs/products_all)
* [products_basicFields](https://java.documentationai.com/docs/products_basicfields)
* [products_keysSourceURLs](https://java.documentationai.com/docs/products_keyssourceurls)
* [products_all_nested](https://java.documentationai.com/docs/products_all_nested)
* [products_pricesFlat](https://java.documentationai.com/docs/products_pricesflat)
* [products_reviewsFlat](https://java.documentationai.com/docs/products_reviewsflat)
