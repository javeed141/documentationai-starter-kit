When using JSON data:

[Web Browser](https://java.documentationai.com/docs/business-data-with-web-browser-and-json) | [Postman](https://java.documentationai.com/docs/business-data-with-postman-and-json) | [cURL](https://java.documentationai.com/docs/business-data-with-curl-and-json) | [PHP](https://java.documentationai.com/docs/business-data-with-php-and-json) | [Node.js](https://java.documentationai.com/docs/business-data-with-nodejs-and-json) | [Python](https://java.documentationai.com/docs/business-data-with-python-and-json)
