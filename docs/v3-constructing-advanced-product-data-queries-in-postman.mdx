<Callout kind="info">
  **The &quot;Wild Card&quot;**

Using ***** in your query will search for any possible value that could fill that spot. In the above case, it is essentially saying "Find all records that have a key value set to anything". **Note that you can not prefix * any string.**
</Callout>

You can query the database using most of the fields in the [Product Data Schema](https://developer.datafiniti.co/v4/docs/product-data-schema).

  **Custom View Formatting**

When creating the `"view"` object, put the entire thing inside of `[ ]`, with each field being called inside of `{ }`. If you are listing multiple fields, put a comma after each one except for the last one. See the screenshot above for an example.
