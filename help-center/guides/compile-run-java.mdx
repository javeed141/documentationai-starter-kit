## Overview

Write, compile, and run your first Java program.

<Steps>
  <Step title="Create the source file">
    ````java
    // HelloWorld.java
    public class HelloWorld {
      public static void main(String[] args) {
        System.out.println("Hello, Java!");
      }
    }
    ````
  </Step>
  <Step title="Compile">
    ````bash
    javac HelloWorld.java
    ````
  </Step>
  <Step title="Run">
    ````bash
    java HelloWorld
    ````
  </Step>
</Steps>
