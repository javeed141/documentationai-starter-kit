## Overview

Install Java Development Kit (JDK) to get started.

## Steps

1. Visit `https://jdk.java.net/`
2. Download the latest JDK for your OS (Windows, macOS, Linux)
3. Run the installer or extract files as prompted
4. Add the JDK `bin` directory to your `PATH` environment variable

## Verify installation

```bash
java -version
```

You should see Java version info displayed.
