## GitHub Version Control Integration

Connect your GitHub repository to automatically publish documentation changes.

<Tabs>
  <Tab title="Connect Repository" icon="link">
    1. Navigate to your Java dashboard at `https://dashboard.example.com`.
    2. Go to **Settings > Integrations > GitHub**.
    3. Authorize the app and select your repository.

    <CodeGroup tabs="CLI,UI">
    ````bash
    gh auth login
    java-docs sync-repo --repo owner/my-docs
    ````
    ````javascript
    // Using Java SDK
    import { JavaDocs } from '@java/docs-sdk';
    const docs = new JavaDocs({ apiKey: 'YOUR_API_KEY' });
    await docs.integrations.connectGitHub('owner/my-docs');
    ````
    </CodeGroup>
  </Tab>
  <Tab title="Sync Workflow" icon="git-branch">
    Use GitHub Actions to trigger builds.

    ````yaml
    name: Deploy Docs
    on: [push]
    jobs:
      deploy:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v4
          - run: java-docs deploy --token ${{ secrets.JAVA_TOKEN }}
    ````
  </Tab>
</Tabs>

## Webhook Setup for Real-Time Updates

Set up webhooks to receive instant notifications on documentation events like page updates or new viewers.

### Steps to Configure

<Steps>
  <Step title="Create Webhook" icon="plus">
    In your Java dashboard, visit **Settings > Webhooks > New Webhook**.

    Provide your endpoint: `https://your-webhook-url.com/java-events`.
  </Step>
  <Step title="Select Events" icon="zap">
    Choose events: `page.updated`, `page.published`, `user.login`.
  </Step>
  <Step title="Verify and Test" icon="check-circle">
    Java sends a test payload. Respond with HTTP 200.

    <ParamField header="X-Java-Signature" param-type="string" required="true">
      HMAC SHA-256 signature for payload verification.
    </ParamField>
  </Step>
</Steps>

Example handler:

<CodeGroup tabs="Node.js,Python">
````javascript
const crypto = require('crypto');
app.post('/java-events', (req, res) => {
  const signature = req.headers['x-java-signature'];
  const payload = JSON.stringify(req.body);
  const expected = crypto.createHmac('sha256', 'YOUR_WEBHOOK_SECRET')
    .update(payload).digest('hex');
  if (signature === `sha256=${expected}`) {
    console.log('Event:', req.body.event);
    res.status(200).send('OK');
  } else {
    res.status(401).send('Invalid signature');
  }
});
````
````python
import hmac
import hashlib
from flask import Flask, request

app = Flask(__name__)
@app.route('/java-events', methods=['POST'])
def webhook():
    signature = request.headers.get('X-Java-Signature')
    payload = request.data
    expected = hmac.new(
        b'YOUR_WEBHOOK_SECRET',
        payload,
        hashlib.sha256
    ).hexdigest()
    if signature == f'sha256={expected}':
        print('Event:', request.json['event'])
        return 'OK', 200
    return 'Invalid signature', 401
````
</CodeGroup>

Build custom integrations using Java's REST API at `https://api.example.com/v1`.

<ResponseField name="data" field-type="object" required="true">

## Embedding External Content

Embed GitHub Gists, YouTube videos, or iframes directly in pages.

| Provider | Embed Method | Example |
|----------|--------------|---------|
| GitHub Gist | `<iframe>` | `https://gist.github.com/...` |
| YouTube | `<iframe>` | `https://youtube.com/embed/VIDEO_ID` |
| Figma | `<iframe>` | `https://www.figma.com/embed?embed_host=...` |

<Callout kind="alert">
  Embedded content must comply with your site's Content Security Policy (CSP).
</Callout>
