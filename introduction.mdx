## What is Java?

Java provides a centralized space for you to organize, manage, and collaborate on all your project documentation. You can create structured docs, embed code examples, and maintain version history effortlessly. Whether you're documenting APIs, user guides, or internal wikis, Java scales with your team's needs.

This platform supports markdown-based editing with live previews, making it easy to author content quickly. Integrate it with your existing tools like GitHub or Jira for seamless workflows.

## Quick Start Overview

Get up and running in minutes with these essential steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.java-docs.com/register` using your email or GitHub account.
  </Step>
  <Step title="New Project" icon="plus">
    Click **New Project** and name it (e.g., "My API Docs"). Add a description for context.
  </Step>
  <Step title="Add First Document" icon="file-text">
    Create a new MDX file and start editing. Use the live preview to see changes instantly.

````mdx
# My First Doc

Welcome to your documentation!
````

  </Step>
  <Step title="Publish & Share" icon="share-2">
    Hit **Publish** to generate a shareable link. Embed it in your README or site.
  </Step>
</Steps>

<Callout kind="tip">
  Pro tip: Use the `/invite` command in your project settings to add collaborators immediately.
</Callout>
