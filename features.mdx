## Document Creation and Editing Tools

Start new documents instantly and edit with a full-featured WYSIWYG or Markdown editor. Java supports syntax highlighting for code blocks, embedded images, and interactive components like tabs and callouts.

<Steps>
  <Step title="Create a New Document" icon="plus">
    Navigate to your workspace and click `New Document`. Choose a template for API docs, guides, or changelogs.
  </Step>
  <Step title="Edit Content" icon="edit">
    Use the editor toolbar for headings, lists, and embeds. Switch to Markdown view for precise control:
    
````markdown
# Welcome

This is your **first document**.

```javascript
console.log("Hello, Java!");
```
````
    
  </Step>
  <Step title="Preview and Publish" icon="eye">
    Preview changes live, then publish to make it available.
  </Step>
</Steps>

<Callout kind="tip">
Save frequently—auto-save runs every 30 seconds, but manual saves ensure no data loss.
</Callout>

## Collaboration and Version Control

Invite team members to co-edit documents in real-time. Java integrates with Git for version control, allowing pull requests and branch previews.

<Tabs>
  <Tab title="Real-time Collaboration" icon="message-circle">
    Share a link with edit permissions. Multiple users see changes instantly, with cursor indicators and comments.
  </Tab>
  <Tab title="Git Integration" icon="git-branch">
    Connect your repo:
    
````bash
git remote add java https://git.example.com/project-docs.git
git push origin main
````
    
    Preview branches before merging.
  </Tab>
</Tabs>

## Search and Organization Features

Powerful full-text search indexes all documents, including code snippets. Organize with folders, tags, and table of contents.

| Feature | Description | Benefit |
|---------|-------------|---------|
| Global Search | Searches titles, content, and code | Find anything in seconds |
| Tags | Add custom labels like `api`, `guide` | Filter and categorize |
| Folders | Nested structure | Logical hierarchy |

Use tags like `{api-endpoint}` in backticks for quick searches.

## Exporting and Sharing Documentation

Export docs as PDF, HTML, or Markdown archives. Share public links with password protection.

<CodeGroup tabs="PDF,HTML,Markdown">
````bash
java export --format=pdf --output=docs.pdf
````

````bash
java export --format=html --output=site/
````

````bash
java export --format=md --output=archive/
````
</CodeGroup>

<Callout kind="success">
Exported docs retain interactive elements like tabs when using HTML format.
</Callout>
